🏢 OpenAI 推出 Frontier 平台：AI 代理人的「人資管理系統」

OpenAI Source: The Verge

管理人類很難，管理 AI 代理人（Agents）也同樣不簡單。這就是為什麼 OpenAI 推出了名為 "OpenAI Frontier" 的全新平台，旨在幫助企業「構建、部署和管理」AI 代理人，即使這些代理人並非由 OpenAI 自身開發。

OpenAI 對 Frontier 的描述聽起來就像是 AI 界的 HR 系統。該公司在部落格文章中寫道：「Frontier 賦予代理人如同人類在職場成功所需的技能：共享的上下文（Context）、入職培訓（Onboarding）、帶有反饋的實作學習，以及清晰的權限與邊界。」這種相似性並非巧合：OpenAI 表示，該產品的靈感正是來自於觀察企業如何擴展人類團隊的規模。

🔥 超級盃廣告戰！Sam Altman 砲轟 Anthropic 廣告「不誠實」

Drama Source: The Verge

OpenAI 執行長 Sam Altman 週三在 X 平台上回擊了競爭對手 Anthropic 的超級盃新廣告，直指其「顯然不誠實」（clearly dishonest），並稱這種「雙言巧語」（doublespeak）很符合 Anthropic 的品牌風格。

Altman 寫道：「我們顯然永遠不會像 Anthropic 描述的那樣投放廣告。我們不笨，我們知道用戶會拒絕這種做法。」Anthropic 的廣告雖然沒有指名道姓，但諷刺了競爭對手（暗示 ChatGPT）在 AI 中插入廣告的做法。Anthropic 最近承諾 Claude 將保持無廣告，這與 OpenAI 計劃在 ChatGPT 中引入廣告形成鮮明對比。

💻 GitHub 原生整合 Claude 與 Codex，開發者有了更多 AI 隊友

Dev Tools Source: The Verge

GitHub 今天宣佈，Anthropic 的 Claude 和 OpenAI 的 Codex AI 程式碼代理人將直接進駐 GitHub。這項新的公開預覽版功能讓擁有 Copilot Pro Plus 或 Copilot Enterprise 訂閱的用戶，能在 GitHub 網頁版、GitHub Mobile 以及 Visual Studio Code 中直接使用 Claude 和 Codex。

此舉是 GitHub "Agent HQ" 願景的一部分，旨在讓 AI 代理人成為開發者日常工作流程的原生部分。現在，開發者在創建任務時可以選擇 Copilot、Claude、Codex 或其他自訂代理人。每個程式碼代理人都將消耗一次付費請求（premium request），開發者可以指派這些代理人來處理 Issue 和 Pull Requests。
