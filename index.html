🚀 Anthropic 發布 Claude Opus 4.6：更強大的金融研究與程式碼能力

Anthropic Source: Bloomberg

Anthropic 於 2026 年 2 月 5 日正式發布其最新旗艦模型 Claude Opus 4.6，這是繼去年 11 月 Opus 4.5 之後的重大升級。新模型專為處理複雜的金融研究任務而設計，能夠深入分析公司數據、監管文件與市場資訊，產出原本需要人類花費數天才能完成的詳細財務分析報告。

除了金融領域，Opus 4.6 在多項工作相關功能上也有顯著提升，包括製作試算表與簡報、以及軟體開發能力。Anthropic 表示，新模型能「更謹慎地規劃、更持久地執行代理任務、在大型程式碼庫中可靠運作，並能自行發現錯誤」。這項發布正值 AI 產業競爭白熱化之際，Anthropic 近期簽署了一筆 100 億美元的融資協議，估值達 3500 億美元。

💻 GitHub Copilot 正式支援 Claude Opus 4.6

GitHub Source: GitHub Changelog

GitHub 官方宣布，Anthropic 最新發布的 Claude Opus 4.6 模型現已在 GitHub Copilot 中正式上線。根據早期測試，Claude Opus 4.6 在 代理式程式開發（agentic coding）方面表現出色，特別擅長處理需要規劃與工具調用的高難度任務。

支援範圍：Copilot Pro、Pro+、Business 與 Enterprise 用戶皆可使用。開發者可在以下環境中透過模型選擇器切換至 Claude Opus 4.6：

  • Visual Studio Code（所有模式：chat、ask、edit、agent）
  • Visual Studio（agent、ask 模式）
  • github.com 網頁版
  • GitHub Mobile（iOS / Android）
  • GitHub CLI
  • GitHub Copilot coding agent（僅限 Pro / Pro+ 帳戶）

注意：Copilot Enterprise 與 Business 方案的管理員需先在 Copilot 設定中啟用 Claude Opus 4.6 政策。目前正在逐步推出中，若尚未看到請稍候。

🔥 超級盃廣告大戰！Anthropic 與 OpenAI 正面交鋒

AI Drama Source: Mercury News / AP

ChatGPT 與 Claude 兩大 AI 聊天機器人背後的公司——OpenAI 與 Anthropic——今年將迎來「生死對決」，雙方都需要證明自己的商業模式能夠實現盈利。這場激烈競爭現已延燒至超級盃廣告戰場。

Anthropic 將在本週日的超級盃期間播出兩支電視廣告，幽默諷刺 OpenAI 開始在免費版與低價版 ChatGPT 中投放數位廣告的做法。廣告以誇張的方式呈現「操控性聊天機器人」如何先與用戶建立關係、再推銷產品的危險，結尾打出標語：「廣告即將進入 AI，但不會進入 Claude。」

OpenAI 執行長 Sam Altman 在社群媒體上回擊，稱這些廣告「有趣但不誠實」，並嘲諷 Anthropic「只服務有錢人」。他強調：「使用免費 ChatGPT 的德州人，比全美國使用 Claude 的人還多。」OpenAI 共同創辦人 Greg Brockman 也加入戰局，質疑 Anthropic 是否真的承諾永遠不出售用戶數據給廣告商。這場公開互嗆凸顯了兩家公司自 2021 年分道揚鑣以來的持續競爭。

📈 AI 三巨頭 IPO 倒數！OpenAI、Anthropic、xAI 估值飆升

IPO Source: CNBC

AI 領域的三大獨角獸正面臨 IPO 的關鍵時刻。根據最新報導，Anthropic 本月稍早簽署了一份 100 億美元融資協議，估值達 3500 億美元；而 OpenAI 據報正瞄準超過 8000 億美元的估值。與此同時，Google 母公司 Alphabet 的股價在過去一年飆漲超過 60%。

Anthropic 的 Claude Code 已成為爆款產品，重創傳統軟體股並迫使 OpenAI 緊急回應——後者隨即推出了 Codex 代理人的更新版本。這場 AI 軍備競賽正在重塑企業軟體市場，讓投資人見識到 AI 改變產業的驚人速度。
